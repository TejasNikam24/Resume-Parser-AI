# 🚀 Resume Parser AI

An AI-powered Resume Parser that extracts structured information from resumes (PDF/DOCX) using Natural Language Processing and Generative AI techniques.

## 📌 Project Overview

Resume Parser AI is designed to automate the process of extracting useful information from resumes such as:

- 👤 Personal Details
- 💼 Work Experience
- 🎓 Education
- 🛠 Skills
- 📊 Projects & Achievements

The system converts unstructured resume data into structured JSON format, making it easier for recruiters and HR systems to analyze candidates efficiently.

- Automate resume screening process

## 🧠 Features

- 📂 Upload Resume (PDF/DOCX)
- 🔍 Extract key information using AI
- 🧾 Structured JSON Output

## ⚙️ How It Works

1. 📥 Upload Resume
2. 📄 Extract Text from File
3. 🧠 Process Text using NLP / AI Model
4. 🔎 Identify Entities (Name, Skills, etc.)
5. 📊 Convert into Structured Format (JSON)
